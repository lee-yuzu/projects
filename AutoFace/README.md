### 🚙 AutoFace — “앞모습만 보고 차종을 알아내는 AI 차량 감별사”

사용자가 차량 이미지를 업로드하면, 해당 차량의 차종을 알려주는 서비스입니다.
추후 중고차 자동 분류, 도로 모니터링 등의 분야에서 활용될 수 있도록 설계되었습니다.
